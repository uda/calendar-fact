# Calendar fact

Taken from [xkcd.com/1930](https://xkcd.com/1930/), this command generates
calendar facts. Useful for MODs, head or foot lines in apps etc.

## Installation

Install by running:
```shell
pip install calendar-fact
```

## Usage in shell

Run:
```shell
calendar-fact
```

## Usage in code

```python
from calendar_fact import get_fact

new_fact = get_fact()
```
